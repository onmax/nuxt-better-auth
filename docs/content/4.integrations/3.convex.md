[Nuxt Convex](https://nuxt-convex.onmax.me/) provides a real-time backend with auto-syncing queries. Combine it with this module for authentication while using Convex for your application data.

## Setup

::steps

### Install Nuxt Convex

```bash [Terminal]
pnpm add nuxt-convex
```

### Configure nuxt.config.ts

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-convex', '@onmax/nuxt-better-auth'],
  convex: { storage: true },
})
```

### Create Server Auth Config

```ts [server/auth.config.ts]
import { defineServerAuth } from '@onmax/nuxt-better-auth/config'

export default defineServerAuth(() => ({
  emailAndPassword: { enabled: true },
}))
```

::

## Using with Convex Queries

Pass the authenticated user ID from Better Auth to your Convex queries:

```vue [app/pages/tasks.vue]
<script setup lang="ts">
import { api } from '#convex/api'

const { user } = useAuth()
const { data: tasks } = await useConvexQuery(
  api.tasks.list,
  computed(() => user.value ? { userId: user.value.id } : undefined)
)
</script>
```

The query re-subscribes automatically when the user changes. When `undefined` is passed, the query pauses and returns no data.

## Convex as Auth Database

Store authentication data directly in Convex instead of a separate SQL database. This keeps all your data in one place with Convex's real-time sync.

### Install Dependencies

```bash [Terminal]
pnpm add @convex-dev/better-auth better-auth --save-exact
```

### Configure Convex App

```ts [convex/convex.config.ts]
import betterAuth from '@convex-dev/better-auth/convex.config'
import { defineApp } from 'convex/server'

const app = defineApp()
app.use(betterAuth)
export default app
```

### Update Server Auth Config

```ts [server/auth.config.ts]
import { components } from '#convex/api'
import { createClient } from '@convex-dev/better-auth'
import { convex } from '@convex-dev/better-auth/plugins'
import { defineServerAuth } from '@onmax/nuxt-better-auth/config'

const authComponent = createClient(components.betterAuth)

export default defineServerAuth(() => ({
  database: authComponent.adapter(),
  plugins: [convex()],
  emailAndPassword: { enabled: true },
}))
```

Users, sessions, and accounts are stored in Convex with real-time sync capabilities.
