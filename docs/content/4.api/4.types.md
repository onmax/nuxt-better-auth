```ts
import type {
  AuthMeta,
  AuthMode,
  AuthRouteRules,
  AuthSession,
  AuthUser,
  RoleName,
} from '@onmax/nuxt-better-auth'
```

### `AuthMeta`

Used in page meta and `routeRules.auth`.

### `AuthRouteRules`

Extends Nitro route rules with:

- `auth?: AuthMeta`
- `role?: RoleName | RoleName[]`

### Module augmentation

Augment `AuthUser` / `AuthSession` by declaring module `#nuxt-better-auth`.
