## Automatic Inference

When you add plugins or modify your Better Auth configuration, the module automatically updates the `AuthUser` and `AuthSession` interfaces. This means you get immediate autocomplete for fields added by plugins.

**Example:**

If you add the `admin` plugin:

```ts [server/auth.config.ts]
import { admin } from 'better-auth/plugins'
import { defineServerAuth } from '@onmax/nuxt-better-auth'

export default defineServerAuth(() => ({
  plugins: [admin()]
}))
```

You can immediately access the `role` property in your Vue components:

```vue
<script setup lang="ts">
const { user } = useUserSession()

// user.value.role is fully typed!
if (user.value?.role === 'admin') {
  // ...
}
</script>
```

## Manual Augmentation

In rare cases where you extend the user object outside of Better Auth's plugin system (e.g., custom database queries merging data), you can manually extend the types.

Create a file `types/auth.d.ts`:

```ts [types/auth.d.ts]
import '#nuxt-better-auth'

declare module '#nuxt-better-auth' {
  interface AuthUser {
    customField?: string
  }
}
```
