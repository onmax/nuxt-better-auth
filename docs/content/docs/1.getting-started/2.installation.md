## Prerequisites

- A Nuxt project (v3.10+)
- [NuxtHub](https://hub.nuxt.com) module installed and configured (for database support)
- A database dialect supported by NuxtHub (SQLite, PostgreSQL, MySQL)

::callout{icon="i-lucide-info" to="/docs/getting-started/quickstart"}
Looking for a complete example? Check out the **Quickstart** guide.
::

## Add to project

::steps

### Install dependencies

Install the module and `better-auth`. You also need `drizzle-orm` as it is used for schema generation.

::code-group
```bash [pnpm]
pnpm add @onmax/nuxt-better-auth better-auth drizzle-orm
```
```bash [npm]
npm install @onmax/nuxt-better-auth better-auth drizzle-orm
```
```bash [yarn]
yarn add @onmax/nuxt-better-auth better-auth drizzle-orm
```
```bash [bun]
bun add @onmax/nuxt-better-auth better-auth drizzle-orm
```
::

### Register Module

Add `@onmax/nuxt-better-auth` to your `nuxt.config.ts`. Ensure `@nuxthub/core` is also present and configured with a database.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: [
    '@nuxthub/core',
    '@onmax/nuxt-better-auth'
  ],

  hub: {
    db: true // or { dialect: 'postgresql' }
  },

  runtimeConfig: {
    betterAuthSecret: '...' // Generate with `openssl rand -hex 32`
  }
})
```

### Create Configuration Files

The module requires two configuration files to link Nuxt with Better Auth.

1. **Server Configuration**: `server/auth.config.ts`
2. **Client Configuration**: `app/auth.client.ts`
